    HTTP 403 → 크롤러 허용으로 서버 설정 즉시 변경

    arkenkomoshnikkosnova

    3개 사이트 모두 외부 HTTP 요청에 403 반환 중. Cloudflare/Nginx 설정에서 Googlebot(UA: "Googlebot"), Bingbot 등 검색 엔진 크롤러 IP 대역을 허용 목록에 추가. 가장 시급하며 이것이 해결되지 않으면 다른 모든 SEO 작업이 무의미함.

    robots.txt 생성 및 sitemap.xml 제출 (Google Search Console)

    arkenkomoshnikkosnova

    각 사이트에 robots.txt 생성 (User-agent: * / Allow: /). XML 사이트맵 생성 후 Google Search Console에 제출. 특히 kosnova.kr은 게시판 URL 패턴을 sitemap에 포함해 최소 50페이지 이상 색인 유도 필요.
